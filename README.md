# Twitter Sentiment Analysis #





## 📌 PROJECT OVERVIEW:

This project performs sentiment analysis on Twitter data to understand public opinion about topics, brands, or products.
It uses Natural Language Processing (NLP) techniques to classify tweets into:

- 😊 Positive
- 😐 Neutral
- 😡 Negative

The results are visualized using Matplotlib to show sentiment trends over time and overall distribution.

## 🎯 OBJECTIVES: ##
- Clean and preprocess Twitter text data
- Perform sentiment analysis using VADER (NLTK)
- Classify tweets into sentiment categories
- Visualize sentiment patterns using graphs
## 🧠 TOOLS USED: ##
- Python 🐍
- Pandas 📊
- NLTK (VADER Sentiment Analysis) 🧠
- Matplotlib 📈
- Regex (Text Cleaning)
- 📂 Dataset
